# Deploy KSports Website

## Vercel
1. Upload project to GitHub
2. Import repository into Vercel
3. Add environment variables from .env.example
4. Deploy

## Netlify
1. Drag and drop folder or connect GitHub
2. Build command:
npm run build

3. Publish directory:
dist

## Install
npm install

## Run Development
npm run dev

## Build Production
npm run build
